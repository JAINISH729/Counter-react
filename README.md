🔢 React Counter App

A simple counter application built using React JS. The app allows users to increase and decrease a number by clicking buttons. This project demonstrates basic React concepts like components, state management, and event handling.

🚀 Features

➕ Increase counter value

➖ Decrease counter value

⚡ Instant UI updates using React state

🧠 Beginner-friendly React project

🛠️ Built With

⚛️ React JS

JavaScript (ES6+)

HTML5

CSS3

📚 Concepts Used

Functional Components

useState Hook

Event Handling

JSX

Component Re-rendering

🎯 How It Works

The counter value is stored in React state using the useState hook.

Clicking the Increase button increments the state.

Clicking the Decrease button decrements the state.

React automatically re-renders the component when the state changes.
